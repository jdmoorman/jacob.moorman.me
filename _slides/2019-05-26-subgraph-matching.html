Find all signals where \\(\mathbf{A} = \mathbf{1}\\) 1. Find all signals where \\(\mathbf{A} = \mathbf{1}\\) and \\(\mathbf{C} = \mathbf{6}\\) 1. ... 2. Find all signals where \\(\mathbf{A} = \mathbf{4}\\) 3. ... -vertical- ## Tree Search * In what order should we match the template nodes? * To which world nodes should we try matching them? -vertical- ## Constraint Propagation **To which world nodes should we try matching them?** Try matching template nodes to world nodes which fulfill a subset of the subgraph isomorphism constraints. -horizontal- ## Our Work * Stats filter * Topology filter * Neighborhood filter * Permutation filter * Public GitHub code -horizontal- ## Future Work
